# API

# Clone

    git clone git@github.com:neo-portfolio/API.git

# Installation

# Install Rust (+ cargo)

Follow instructions on [the official website.](https://www.rust-lang.org/tools/install)

    
## Compile

### Debug

    cargo run --color=always --package api --bin api
    
### Production
    
    cargo build --release
    
    
# Run

    cargo run
